# How to Install

## Requirements

1. [Install Conda](https://docs.conda.io/projects/miniconda/en/latest/miniconda-install.html).
2. Conda env with `python=3.10`, `git` and `pip` installed. You can do this by running this command after installing Conda:

<CodeGroup>
  
  ```bash {{ title: 'cURL' }}
  conda create --name glair-genai-sdk python=3.10 pip git conda activate
  glair-genai-sdk
  ```

</CodeGroup>
